1、C语言宏中的 # 和 ## 
# 可以将宏定义中的传入参数名转换为双引号括起来的参数名字字符串，必须放在宏定义的参数名前
## 将宏定义的多个形参转换成一个实际的参数名

2、关键词 volatile 的使用场景 
volatile (直接存取原始内存地址)、避免对应的变量值在程序不知道的情况下发生改变
使用场景：
    1、并行设备的硬件寄存器（状态寄存器）
    2、中断服务子程序会访问的变量
    3、多线程应用中被几个任务共享的变量

3、关键词 static 的作用
作用 ： 更改函数与变量的作用域，只在内存中分配一次
1、在函数体内，一个被声明的静态局部变量在函数反复调用时保持值不变
2、模块内，函数体外，一个被声明的全局静态变量只能被模块内函数访问
3、模块内的静态函数可以被模块内的其他函数使用

4、extern 关键字的作用
    1、变量声明情况
        int i； 声明  + 定义 ，需要存储空间
        extern int i；  声明 ，不分配存储空间
    2、在C语言中，修饰符 extern 用在变量和函数的声明之外，用此来说明 变量函数 是在别处定位的，需要在这里引用

    3、extern “C” 表示这段代码要按C语言规则编译

```C
#ifndef __INCvxWorksh /*防止该头文件被重复引用*/
#define __INCvxWorksh
#ifdef __cplusplus//告诉编译器，这部分代码按C语言的格式进行编译，而不是C++的
extern "C"{
#endif
 
/*…*/
 
#ifdef __cplusplus
}
 
#endif
#endif /*end of __INCvxWorksh*/
```

5、 const 关键字的作用
    1、定义变量为常量，必须初始化；
    2、修饰函数的参数，表示在函数体内不能修改这个参数的值
    3、修饰函数的返回值，只能用于修饰指针变量，一般是指修饰的内容不变
    
6、new/delete 和 malloc/free 的区别
    1、new/delete 是C++ 中的操作符， malloc 和 free 是库函数
    2、new/delete 对应于构造函数和析构函数， malloc / free 是内存分配


7、strlen("\0")=0 和 sizeof("\0") =2的联系与区别
    1、strlen是库函数，用于计算字符串长度()，遇到\0结束。sizeof是关键字，以字节形式给出操作数存储大小。
    2、strlen在运行期计算，sizeof在编译期计算。

8、C语言是怎么进行函数调用的
    CPU的程序是通过栈来实现函数调用的，栈被用于传递函数参数，存储返回信息，临时保存寄存器原有的值以备回复，用来存储局部变量
    函数调用操作所使用的栈部分叫做栈帧结构
