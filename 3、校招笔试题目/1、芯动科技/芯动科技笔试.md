# 1.3
求整数 n（n>=0）阶乘的算法如下，其时间复杂度是 ==（D）==
```C
int fact(int n) {
    if (n <= 1) {
        return 1;
    }
    return n * fact(n - 1);
}
```
1. O(logn)
2. O(nlogn)
3. O(n^2)
4. O(n)
解答：
该函数用于计算 n!，需要循环 n 次，每次循环执行一次乘法操作，且乘法运算的复杂度是常数时间 O(1)，时间复杂度为 O(n)，选择选项 D

# 3.1
自旋锁和互斥锁的区别及应用场景
解答：
1、常见的同步机制，用于对共享资源的保护
2、自旋锁在获取锁失败时不会使线程进入休眠，而是持续不断地循环（占用CPU）检查锁是否可用
3、互斥锁在获取锁失败时会使线程进入休眠状态，等待锁可用是唤醒线程
4、自旋锁一般用在短时间持有的情况或者中断上下文中
5、互斥锁一般用在长时间持有锁的情况或者用于线程同步的情况

# 3.2 
什么事内联函数，一般用在什么情况下，要注意什么
解答：
1、内联函数是使用 inline 关键词修饰的函数，内联函数在编译时编译器会直接展开该函数，不会产生函数调用时的参数以及返回值的开销，同时也不会产生上下文保存所需要的开销
2、一般的使用情况：
适用于小型、简单且会被频繁调用的函数
3、注意事项：
不能将递归函数定义为内联函数，同时应该避免将函数体复杂、庞大的函数定义为内联函数，否则会导致代码膨胀

# 4.1
删除有序链表中重复的元素
给出一个升序排序的链表，删除链表中的所有重复出现的元素，只保留原链表中只出现一次的元素
例如：
给出的链表为 1->2->3->3->4->4->5，返回 1->2->5
给出的链表为 1->1->1->2->3，返回 2->3
数据范围：
链表长度 0 < n < 10000，链表中的值满足 | val | < 1000
要求：
空间复杂度不大于 O(n)
时间复杂度不大于 O(n)
解答：
```C

// 定义一个链表节点
struct ListNode {
    int val;
    struct ListNode *next;
};
// 删除有序链表中重复的元素
struct ListNode* deleteDuplicates(struct ListNode* head) {
    if (head == NULL || head->next == NULL) {
        return head;
    }
    struct ListNode *p = head;
    while (p->next != NULL) {
        if (p->val == p->next->val) {
            struct ListNode *temp = p->next;
            p->next = temp->next;
            free(temp);
        } else {
            p = p->next;
        }
    }
    return head;
}
```
