# 编写字符驱动程序
1、确定主设备号
2、创建file_operation结构体
    在里面填充 drv_open\drv_read\drv_ioctl等函数
3、注册file_operation 结构体
    register_chrdev(major,&fops,name)
4、写退出函数 
    unregister_chrdev(major,name)

i2c_driver表明能支持那些设备：
    使用 of_match_table 来判断
        设备树中，某个 I2C 控制器节点下可以创建I2C 设备节点
        如果 I2C 设备节点的 cmpatible 属性跟 of_match_table 中的匹配
        则该设备节点会被 i2c_driver 支持
# 示例
1、编写一个字符驱动程序，支持 I2C 设备
static struct i2c_driver at24_driver = {
    .driver = {
        .name = "at24", //能支持哪些设备（主要用来判断设备树中的I2C设备）

        .acpi_match_table = ACPI_PTR(at24_acpi_ids),
    },
    .probe_new = at24_probe,    //找到能支持的设备后，该函数被调用
    .remove = at24_remove,      //设备节点被移除时，该函数被调用，
    .id_table = at24_id_table,  //设备树中，某个 I2C 设备节点的 id 属性，
