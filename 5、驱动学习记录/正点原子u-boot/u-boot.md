    2.1 信息查询命令    
             bdinfo : 打印板信息

    3.1 setenv ： 设置环境变量

    3.2 saveenv ： 保存环境变量到存储

    3.3 setenv : 自定义环境变量


2、u-boot 内存与网络命令

    1.1 md 命令 用于显示内存值
        md[.b,.w,.l] address 数量 // b表示一个字节、w表示两个字节、l表示四个字节 ; address表示查看的内存起始地址，数量表示长度，这个数据的长度单位不是字节和所选择的格式有关

    1.2 nm 命令 用于修改指定地址的内存值 
        nm [.b .w .l] address 
        比如 我想在以 .l 格式修改 0x80000000 地址的数据为 0x12345678 
        nm .l 80000000
        他会跳出来一个当前值，直接在后面输入就行

    1.3 mm 命令 用于修改内存值，用 mm 命令修改内存值时地址会自增，但是 nm 命令的地址不会自增
    比如 以 .l 格式修改从地址 0x80000000 开始的连续三个内存块的数据为 0x05050505 
    mm .l 80000000

    1.4 mw命令 ： 用于使用一个指定的数据填充一段内存，命令格式如下：
    mw [.b .w .l] address value [count]
    比如 使用 .l 格式将以 0x80000000 为起始地址的 0x100 个内存块填充为 0x0a0a0a0a ,命令如下
    mw .l 80000000 0a0a0a0a 100
    md .l 80000000 100 填充完了再使用md命令查看

    1.5 cp命令
    cp命令式数据拷贝命令，将 DRAM 中的数据从一段内存拷贝到另一段内存中，或者把 Nor flash 中的数据拷贝到 DRAM 中
    cp [.b .w .l] source target count

    1.6 cmp 命令
    比较命令，用于比较两段内存的数据是否相等
    cmp [.b .w .l] addr1 addr2 count
