# 数据类型
## 用变量a 给出对应的定义
- 整形数 int a;
- 指向整形数的指针 int *p;
- 指向指向整形数的指针的指针 int **pp;
- 一个有10个整形数的数组 int a[10];
- 一个有10个指针的数组，该10个指针是指向整形数的指针 int *p[10];
- 数组指针 int (*p)[10];
- 函数指针 int (*p)(int, int);
- 函数指针数组 int (*p[10])(int, int);

- 有符号数会转换成无符号数

## 浮点数与零值的比较语句
- 计算机在处理浮点数的时候是有误差的，不能将浮点类型变量用 “==”或 “！=”与数字进行比较，应该设法转成 < 或者 > 符号

```c
if(x > -0.000001 && x < 0.000001)
```

## 结构体内存对齐原则
1、第一个成员的首地址（offset=0）
2、成员对齐：每个成员的起始地址必须是其类型大小的整数倍。
3、结构体总大小必须是其最大成员大小的整数倍。

## 内存对齐原因
1、平台原因：不是所有的硬件平台都可以访问任意地址上的任意数据的；某些硬件平台只能在某些地址处取某些特定类型的数据，否则抛出硬件异常。
2、性能原因：数据结构（尤其是栈）应该尽可能地在自然边界上对齐。

## 数组首元素地址和数组地址的异同
- 相同点是两者值相同
- 不同点是 数组首元素地址是数组中第一个元素的地址，数组地址是数组本身的地址
